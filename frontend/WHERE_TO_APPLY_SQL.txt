SQL TO RUN (you can copy from here too):

ALTER TABLE tasks 
ADD COLUMN IF NOT EXISTS progress_percentage INTEGER DEFAULT 0 
CHECK (progress_percentage >= 0 AND progress_percentage <= 100);

CREATE INDEX IF NOT EXISTS tasks_progress_percentage_idx 
ON tasks(progress_percentage);

═══════════════════════════════════════════════════════════════════

WHAT THIS DOES:
✅ Adds the missing "progress_percentage" field to your tasks table
✅ Makes sure values are between 0 and 100
✅ Creates an index for faster queries
✅ Enables the progress bar to work!
